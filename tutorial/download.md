# Download

Download code.tar.gz (~70M) or code_noseg.tar.gz (~1M), and set up the environment using Miniconda.

To install offline without Conda, also download environment.tar.gz (~2.7G) or environment_noseg.tar.gz (~500M).

## Code packages

[MPicker_code_v1.3.0.tar.gz](http://thuem.net/website/MPicker/Packages/MPicker_code_v1.3.0.tar.gz) (70M)

[MPicker_code_noseg_v1.3.0.tar.gz](http://thuem.net/website/MPicker/Packages/MPicker_code_noseg_v1.3.0.tar.gz) (1M)

[OptCuts_MPicker_v1.2.0.tar.gz](http://thuem.net/website/MPicker/Packages/OptCuts_MPicker_v1.2.0.tar.gz) (6M)
